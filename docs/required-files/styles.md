# 2.2 Estilos

- ./src/theme

  - [common.ts](https://reactnative.dev) -> estilos globales
  - [colors.ts](https://reactnative.dev) -> colores de la app
  - [fonts.ts](https://reactnative.dev) -> tipografia
  - [responsive.ts](https://reactnative.dev) -> responsive y escalado
